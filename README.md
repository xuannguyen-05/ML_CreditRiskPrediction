# Dự đoán rủi ro tín dụng

Đây là đồ án môn học, xây dựng một mô hình Machine Learning (Logistic Regression)
để dự đoán khả năng vỡ nợ của khách hàng và triển khai mô hình đó
lên một ứng dụng web bằng Flask.

## Chức năng

- Form nhập 16 trường thông tin khách hàng.
- Áp dụng logic "Đèn giao thông" (Xanh/Vàng/Đỏ) để hỗ trợ ra quyết định.
- Giao diện định dạng tiền tệ (VNĐ) tự động.

## Cách chạy

1. Cài đặt các thư viện:
   `pip install -r requirements.txt`

2. Chạy ứng dụng:
   `python app.py`

3. Truy cập: `http://127.0.0.1:5000`
